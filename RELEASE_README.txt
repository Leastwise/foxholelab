## FoxholeLab Pre-1.0 Pre-Release Demo

To run this project locally:

1. Install [NodeJS](https://nodejs.org/en/)
2. Unzip foxholelab
3. Double-click install.bat
4. Double-click run.bat
5. Refresh the browser window that pops open

Make sure you are on the "Satisfactory" tab. Have fun poking around, remember that 
values will likely change after DevBranch!
